## 为什么你需要这个工具？

xStocks 的积分机制高度依赖**每天不断签**：

1. **转盘** — 每天可以对前一天的所有积分抽 **0~10 倍**加成。漏转一天 = 白白丢掉一天的翻倍机会。
2. **连签加成** — 连续签到每天+0.1x，最高 **2x**（11天满）。断签一天 = 加成归零，从头累积。

> 哪怕你持有再多 xStocks，只要漏签一天，你就损失了转盘翻倍 + 连签加成，积分大打折扣。

本工具自动完成每日签到 + 转盘，启动后无需手动操作，保证你一天都不会漏掉。

## 第一步：配置私钥

1. 找到文件夹中的 `.env.example` 文件
2. 复制一份，改名为 `.env`
3. 用记事本打开 `.env` 文件
4. 填入你的钱包私钥：

**MetaMask 等 EVM 钱包：**
```
PRIVATE_KEY=0x1234abcd...（以0x开头的一长串）
```

**Phantom 等 Solana 钱包：**
```
SOL_PRIVATE_KEY=4wBqpZM9xaSh...（一串字母数字）
```

两种钱包**选一种填**就行，另一种保持原样不用管。

> ⚠️ 私钥非常重要，不要发给任何人！

### 如何导出私钥？

- **MetaMask**：打开 MetaMask → 点右上角三个点 → 账户详情 → 显示私钥 → 输入密码 → 复制
- **Phantom**：打开 Phantom → 设置 → 安全和隐私 → 导出私钥 → 输入密码 → 复制

## 拿分策略

### 一、零撸签到（零成本）
- **风险：无**
- 每天运行本工具自动签到 + 转盘，赚取基础积分
- 分数较少，但完全零成本，坚持连签可累积加成倍数（最高2x）

### 二、购买持有 xStocks（1x 积分权重）
- **风险：低**（仅股票价格波动）
- 买入 xStocks 代币放在钱包里，自动计算积分
- 推荐买 SP500（标普500指数）或 Gold（黄金），波动小

EVM链购买平台：
- [1inch](https://1inch.io)
- [Cow Swap](https://swap.cow.fi)
- [Bebop](https://bebop.xyz)

> 注意：EVM链目前仅支持购买持有，不支持存入和组LP

### 三、存入借贷（4x 积分权重）⭐推荐
- **风险：中**（合约风险，只存不借则风险低）
- 将 xStocks 存入 [Kamino](https://kamino.com) 做抵押品
- 只存入不借出的话风险很低，积分权重是持有的 **4倍**
- 仅支持 Solana 链

### 四、提供流动性 / 组LP（7x 积分权重）
- **风险：高**（无常损失 + 合约风险）
- 在 Raydium、Orca 等 DEX 上为 xStocks 提供流动性
- 积分权重最高，但有无常损失风险，价格波动越大亏损越多
- 仅支持 Solana 链，不熟悉LP的不建议操作

### 总结

| 策略 | 权重 | 风险 | 推荐 |
|------|------|------|------|
| 零撸签到 | - | 无 | 所有人 |
| 购买持有 | 1x | 低 | 小资金 |
| 存入借贷 | 4x | 中 | ⭐推荐 |
| 组LP | 7x | 高 | 需要经验 |

> 推荐使用 Solana 链，因为目前只有 Solana 链支持完整的存入借贷和组LP功能，EVM链仅支持购买持有。
